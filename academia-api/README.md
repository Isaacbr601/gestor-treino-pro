# 🏋️ Gestor de Treino Pro

Um projeto pessoal desenvolvido para explorar o ecossistema **Java + Spring Boot** e colocar em prática os conceitos de **Banco de Dados** que estou aprendendo no curso de ADS na FATEC.

## 🧠 O que estou aprendendo com este projeto:
* **Backend:** Como criar uma API real usando Java 25 e Spring Boot 4.
* **Persistência:** Conectar o código com um banco de dados **PostgreSQL** (JDBC/JPA).
* **Segurança:** Uso de variáveis de ambiente para não expor senhas no GitHub.
* **Frontend:** Interface simples em HTML/JS para gerenciar os treinos.

## 🛠️ Como rodar o projeto
1. Configure a variável de ambiente `DB_PASSWORD` com a senha do seu PostgreSQL.
2. Certifique-se de que o banco `academia_db` está criado.
3. Execute a classe `AcademiaApiApplication.java` no IntelliJ.
